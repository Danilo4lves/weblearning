MAIS SÍMBOLOS

CSS

font-weight: quantidade de negrito (entre 100-900)
font-family: fonte utilizada
font-size: tamanho da fonte
text-decoration: underline, overline, line-through
text-align: left, center, right, justify
text-indent: quantidade de pixels (ex.: 50px)
background-color: cor desejada em inglês, ou usando rgb(quantidade red, q. green, q. blue) ou rgba(red,green,blue,transparencia ou alpha entre 0 e 1)
background-image: url("enderecodaimagem");

Tags

Headers:

h1 h2 h3

testeh1

testeh2

testeh3

Links: <a href="endereco">nome do link<a>
Imagem: <img src="endereco" alt="texto para mostrar caso imagem indisponível">
Negrito: <b>teste</b>
Itálico: <i>teste</i>
Itálico2 (enfatizar): <em>teste</em>
Palavra riscada: <del>teste</del>
Underline: Somente através do CSS text-decoration: underline;

Formatação

Sup: <sup>teste</sup>
Subscrito: <sub>teste</sub>
Pré-formatado: <pre>texto pre formatado</pre>
Visualizar códigos: <code>codigo</code> (normalmente utilizado junto com <pre> para deixar pré-formatado)

Comandos semânticos

Alterar visualmente pequenas linhas de texto: <span>texto texto</span>
Quebra de linha: <br>
Espaço em branco: &nbsp;
Quebrar palavra colocando hífen: &shy;
Quebrar palavra sem hífen: <wbr>
